Questions of Exercise 6.1:

*In the code:

	(option == JOptionPane.YES_OPTION ? "Yes" : "No")

+ This is the ternary operator in Java:

	condition ? valueIfTrue : valueIfFalse

+ Therefore:
	- The condition is (option == JOptionPane.YES_OPTION).
	- If the condition is true (user clicks Yes) → it returns "Yes".
	- If the condition is false (user clicks No, Cancel, or closes the dialog) → it always returns "No".
→ Any choice other than Yes will be treated as No in the output.
